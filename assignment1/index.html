Movie Archive

This webpage is going to be a place where you can find the current availability of many movies; so where to watch them.

The different platforms

The following streaming services are the ones I want to cover:

  • Netflix
  • Amazon Prime
  • Disney+

The whole sense about this website (background)

The problem nowadays often lies in having all the streaming services mentioned above cover the movies, but in a flexible status. So one day movie "xyz" is available on Netflix; the next day it is not anymore, but now on Disney+. Currently the way to go is to open Google and search for "Movie "XYZ" Stream", and you will probably find out where the movie is available.

Still, I would like to have the ability to see where a movie is available in one tool, where I can also look for movies to watch. Another idea is the ability to make a watchlist yourself or with your friends, so you have a tool where you can find your watchlist and directly see if a movie is currently available on one of the services or not.

I'm not 100% sure if this is going to be my final project, but I could imagine it; as you can read in my explanation above, this could have a real use for me, to save a watchlist with my friends and directly see which of these movies are available. I can also imagine including a rating feature, but I will have to see (as this tool also somehow exists, but I have not made use of it myself, so I can try to make my own one).

Implementations I expect

These are the features I have in mind:

  1. Watchlist feature
  2. Loading the movies through an API
  3. Rating that should be saved by a server
  4. Availability (on which platform)
  5. Either all movies are already available and the user can add them to a watchlist, or a prepared form where you put movie details in
  6. A direct link to the YouTube trailer of the movie
  7. Either just a rating (1 to 5 stars) or also comments
